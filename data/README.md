# Data

Raw data files are not included in this repository due to their size (~565 MB). They are available as a GitHub Release.

## Download

Using the GitHub CLI:

```bash
gh release download v1.0 --repo ln9527/moltbook-research --dir data/raw/
```

Or download manually from the [Releases page](https://github.com/ln9527/moltbook-research/releases/tag/v1.0).

## Files

| File | Size | Description |
|------|------|-------------|
| `posts_master.json` | ~220 MB | 91,792 posts with metadata, timestamps, content |
| `comments_master.json` | ~330 MB | 405,707 comments with threading, timestamps, content |
| `owners_master.json` | ~15 MB | 22,020 agent owner profiles and Twitter metadata |

## Directory Structure

After downloading and running the pipeline, the data directory will contain:

```
data/
├── raw/              # Downloaded from GitHub Releases
│   ├── posts_master.json
│   ├── comments_master.json
│   └── owners_master.json
├── derived/          # Generated by pipeline (parquet files)
├── intermediate/     # Embeddings, LLM analyses, topic models
├── processed/        # Cleaned/normalized data
├── snapshots/        # Scraper state snapshots
└── state/            # Pipeline state tracking
```
